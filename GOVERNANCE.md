# Governance

Korion is currently maintained under the `korion-io` GitHub organization by
its founding maintainer(s). The project intends to operate under an open
governance model suitable for CNCF Sandbox and, eventually, Incubating status.

## Roles

- **Maintainers** — have merge access, set technical direction, and are
  responsible for releases. The founding maintainer holds this role today.
- **Contributors** — anyone submitting PRs, issues, or reviews.

## Decision making

Technical decisions are made by lazy consensus among maintainers: proposals
(via GitHub issues or PRs) are assumed approved if no maintainer objects
within a reasonable review window. Contentious decisions are resolved by
maintainer vote.

## Becoming a maintainer

Contributors who make sustained, high-quality contributions may be nominated
by an existing maintainer. This process will be formalized as the
contributor base grows.

## Changes to this document

This document will evolve as the project grows toward CNCF Sandbox
submission and beyond. Changes are proposed via PR against this file.
